I'm a **Full Stack Developer** with a strong focus on **Back-end Development**.

I'm an **IT Technician** graduated from **IFSULDEMINAS** and completed **Harvard's CS50 – Introduction to Computer Science**, where I strengthened my knowledge of algorithms, data structures, memory management, SQL and software fundamentals.
